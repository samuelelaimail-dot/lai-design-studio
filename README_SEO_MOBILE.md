# SEO and mobile optimisation

Implemented:
• unique titles and meta descriptions for every public page
• canonical URLs for laidesign.studio
• Open Graph and Twitter Card metadata
• JSON LD structured data for Person, Organization, WebSite and page level entities
• robots.txt and sitemap.xml
• PWA manifest update
• social preview image
• icon assets
• image width and height attributes to reduce layout shift
• lazy loading and async decoding for images
• mobile CSS refinements for nav, hero, buttons, rows and image layouts
• content visibility for below the fold sections
• preloaded logo asset

Note:
The site uses a client side language switcher. For maximum multilingual SEO later, create separate URL paths such as /it/ and /en/ with server rendered language specific HTML and hreflang links.
