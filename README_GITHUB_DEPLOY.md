# GitHub deployment

This folder is ready to push to GitHub Pages.

Important:
- `index.html` must remain at the root of the repository.
- `assets/` must remain at the root.
- `CNAME` is set to `laidesign.studio`. Remove or edit it if you do not want to use that custom domain.
- In GitHub Pages settings, publish from `main` branch and `/root`.

Recommended repository name:
- `lai-design-studio`
- or `laidesign.studio`

After upload:
Repository → Settings → Pages → Build and deployment → Source: Deploy from a branch → Branch: main → Folder: /root → Save.
